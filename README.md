# 山东大学青岛校区绿化战略图管理系统 🌿

这是一个用于校园绿化区域管理的可视化网页系统，支持绘制、编辑和管理绿化区域，实现浇水提醒、状态颜色标识、任务日历、操作日志记录等功能。

## 🚀 功能概览

- 可视化绿化地图（支持绘制区域）
- 自动计算状态（正常 / 需要浇水 / 已逾期）
- 绿化区域详情与浇水频率设置
- 浇水记录及历史分页
- 校园绿化任务日历（自动生成）
- 模态框操作交互（删除、添加、绘制密码保护）

## 🛠️ 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动本地开发服务器

```bash
npm start
```

会自动打开浏览器，访问 `http://127.0.0.1:8080`，即可使用系统。

> 本项目使用 [`live-server`](https://www.npmjs.com/package/live-server) 快速开发预览。
